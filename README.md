# threejs-vr-cardboard

This example shows you how to get a full screen mode window and vr mode for google cardboard by clicking a button

## Requirements

* Nodejs v18+
* Vite
* Three

## Install

* npm install

## Run for development

* npm run dev

## Compile for production

* npm run build
